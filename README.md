# AI-assisted writing in chemistry preprints (2017-2026)

A full-text scientometric study of AI-assisted writing across the ChemRxiv
preprint corpus: prevalence over time, where in the paper it appears, who
adopts it, and what it changes.

This repository is the public research artifact for the study. As of
2026-06-10 it holds the **registered analysis plan**
([`ANALYSIS_PLAN.md`](ANALYSIS_PLAN.md)), committed before the registered
confirmatory analyses were run and before any detector score of the
validation corpus existed. Analysis code, the published per-paper dataset,
and figures will be released here when the preprint is posted. Deviations
from the plan, if any, will be logged in the plan file with dates.

## AI-assistance disclosure

Portions of this study's code and documentation were produced with AI
assistance (Claude, Anthropic). All analyses, design decisions, and final
